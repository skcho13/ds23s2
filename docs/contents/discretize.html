Categorical type in pandas

  • 일종의 데이터 압축 기술이며, 적은 levels로만 이루어진 데이터의 경우 메모리 절약
  • 순서가 부여되어, sorting을 하거나 min/max 함수가 알파벳 순서가 아닌 부여된 순서를 적용
  • 다른 Python library들 중에는 이 순서를 활용하는 것이 있음; 통계적 분석이나 플랏을 그릴 때
